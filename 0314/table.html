scope

th의 scope 값에 'col', 'row'를 입력하면 시각적 어려움을 겪는 사용자의 이해를 도울 수 있다 (+웹 접근성 향상)

Table scope 입력 예제
thead th scope ="col" thead th scope ="col" thead th scope ="col" thead th scope ="col"
tbody scope="row" td td td
tbody scope="row" td td

id - header

제목 셀이 2줄 이상이 되어 복잡한 경우 id="NAME"을 header="NAME" 동일한 값을 입력해서 연결 (+웹 접근성 향상, 실무에서는 scope를 더 많이 사용.)

id, header 입력 예제
thead th thead th id="col-1-1" thead th id="col-1-2"
thead th id="col-2-1" thead th id="col-2-2" thead th id="col-2-3"
tbody th id="row-1" td header="col1-2 col2-2 row-1 td header="col1-1 col1-2 row-1 td header="col1-3 col1-3 row-1
