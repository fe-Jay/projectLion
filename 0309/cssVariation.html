CSS 변수 사용

Calc()

100%에서 좌우 여백을 제거하고 4로 나눠라

Contents width:100%
  • A width:calc(100% / 4)
  • B width:calc(100% / 4)
  • C width:calc(100% / 4)
  • D width:calc(100% / 4)

min(), max(), clamp()

width를 40%로 설정하지만 최대 600px 이하(min)로 제한

min() width: min(40%, 600px)

width를 60%로 설정하지만 최소 200px 이상(max)으로 제한

max() width: max(200px, 50%);

clamp()로 반응형 폰트사이즈 구현

  • A font-size: clamp(0.5rem, 2.5vw, 3rem);
  • B font-size: clamp(0.5rem, 2.5vw, 3rem);
  • C font-size: clamp(0.5rem, 2.5vw, 3rem);
  • D font-size: clamp(0.5rem, 2.5vw, 3rem);
  • E font-size: clamp(0.5rem, 2.5vw, 3rem);
